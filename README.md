 # Digital Tools Buying Website


## Description 
It is e-commerce website where purchasing digital product . This website give interactive experience with cart functionality where product add to cart and deleted product from cart . 


## Technologies Used
- ⚛️ React.js
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🧠 JavaScript (ES6+)
- 🔔 React-Toastify (NPM Package)
- 🗂️ JSON (for managing product data)



## Features about my project

1. Smart Toggling: Product section to cart section
2. Interactive Product & Cart System : Buy Now” button adds product to cart instantly
3. Proceed to checkout button will remove all the products from the cart
